# Keycloak Admin Configuration Page\n\nGuide for implementing an admin page to configure Keycloak server settings.\n\n## Overview\n\nThis admin page allows administrators to:\n- Configure Keycloak server URL, realm, and credentials\n- Test Keycloak connection\n- Configure and test Google social login\n- Configure and test GitHub social login\n- View configuration history\n- Manage session timeout settings\n\n## Architecture\n\n### Backend (tRPC Router)\n\n**File**: `server/routers/keycloak-config.ts`\n\n```typescript\nexport const keycloakConfigRouter = router({\n  // Get current configuration\n  get: adminProcedure.query(async () => { ... }),\n\n  // Update configuration\n  update: adminProcedure.input(KeycloakConfigSchema).mutation(async ({ input, ctx }) => { ... }),\n\n  // Test Keycloak connection\n  testConnection: adminProcedure.input(KeycloakConfigSchema).mutation(async ({ input }) => { ... }),\n\n  // Test social providers\n  testSocialProviders: adminProcedure.input(...).mutation(async ({ input }) => { ... }),\n\n  // Get configuration history\n  getHistory: adminProcedure.query(async () => { ... }),\n});\n```\n\n### Frontend (React Component)\n\n**File**: `client/src/pages/admin/KeycloakConfigPage.tsx`\n\nPage structure:\n1. Configuration form (URL, realm, credentials)\n2. Session settings (timeout)\n3. Social provider configuration\n4. Test buttons and status indicators\n5. Configuration history\n\n## Implementation Steps\n\n### Step 1: Create Backend Router\n\n```typescript\n// server/routers/keycloak-config.ts\nimport { z } from \"zod\";\nimport { adminProcedure, router } from \"../_core/trpc\";\nimport { TRPCError } from \"@trpc/server\";\nimport axios from \"axios\";\n\nconst KeycloakConfigSchema = z.object({\n  url: z.string().url(\"Invalid Keycloak URL\"),\n  realm: z.string().min(1, \"Realm is required\"),\n  clientId: z.string().min(1, \"Client ID is required\"),\n  clientSecret: z.string().min(1, \"Client Secret is required\"),\n  sessionTimeoutMinutes: z.number().int().min(5).max(10080).default(1440),\n  googleClientId: z.string().optional(),\n  googleClientSecret: z.string().optional(),\n  githubClientId: z.string().optional(),\n  githubClientSecret: z.string().optional(),\n});\n\nexport const keycloakConfigRouter = router({\n  get: adminProcedure.query(async () => {\n    // Load from environment or database\n    return {\n      url: process.env.KEYCLOAK_URL,\n      realm: process.env.KEYCLOAK_REALM,\n      clientId: process.env.KEYCLOAK_CLIENT_ID,\n      sessionTimeoutMinutes: 1440,\n    };\n  }),\n\n  update: adminProcedure\n    .input(KeycloakConfigSchema)\n    .mutation(async ({ input, ctx }) => {\n      // Validate by testing connection\n      const testResult = await testKeycloakConnection(input);\n      if (!testResult.success) {\n        throw new TRPCError({\n          code: \"BAD_REQUEST\",\n          message: `Connection test failed: ${testResult.error}`,\n        });\n      }\n\n      // Save to database or environment\n      // In production, save to keycloak_configs table\n\n      return { success: true, message: \"Configuration updated\" };\n    }),\n\n  testConnection: adminProcedure\n    .input(KeycloakConfigSchema)\n    .mutation(async ({ input }) => {\n      return await testKeycloakConnection(input);\n    }),\n\n  testSocialProviders: adminProcedure\n    .input(z.object({\n      url: z.string().url(),\n      realm: z.string(),\n      googleClientId: z.string().optional(),\n      githubClientId: z.string().optional(),\n    }))\n    .mutation(async ({ input }) => {\n      // Test Google and GitHub endpoints\n      return {\n        google: { configured: !!input.googleClientId, available: true },\n        github: { configured: !!input.githubClientId, available: true },\n      };\n    }),\n});\n\nasync function testKeycloakConnection(config: any) {\n  try {\n    const issuerUrl = `${config.url}/realms/${config.realm}`;\n    const response = await axios.get(\n      `${issuerUrl}/.well-known/openid-configuration`,\n      { timeout: 5000 }\n    );\n    return { success: response.status === 200 };\n  } catch (error) {\n    return {\n      success: false,\n      error: error instanceof Error ? error.message : \"Unknown error\",\n    };\n  }\n}\n```\n\n### Step 2: Register Router in Main Routes\n\n**File**:
