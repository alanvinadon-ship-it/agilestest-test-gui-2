# Social Login Quick Start (5 Minutes)\n\nGet Google and GitHub social login working in 5 minutes.\n\n## Prerequisites\n\n- Keycloak running (see KEYCLOAK_MIGRATION.md)\n- Google Cloud account\n- GitHub account\n\n## 1. Get Google Credentials (2 minutes)\n\n1. Go to [Google Cloud Console](https://console.cloud.google.com/)\n2. Create new project: `AgilesTest`\n3. Enable Google+ API\n4. Create OAuth 2.0 credentials (Web application)\n5. Add redirect URI:\n   ```\n   http://localhost:8080/realms/agilestest/broker/google/endpoint\n   ```\n6. Copy **Client ID** and **Client Secret**\n\n## 2. Get GitHub Credentials (1 minute)\n\n1. Go to [GitHub Developer Settings](https://github.com/settings/developers)\n2. Create new OAuth App\n3. Set Authorization callback URL:\n   ```\n   http://localhost:8080/realms/agilestest/broker/github/endpoint\n   ```\n4. Copy **Client ID** and **Client Secret**\n\n## 3. Configure Keycloak (2 minutes)\n\n### Add Google\n\n1. Keycloak Admin: http://localhost:8080/admin\n2. Select realm: `agilestest`\n3. Identity Providers → Create → Google\n4. Paste:\n   - Client ID\n   - Client Secret\n5. Save\n\n### Add GitHub\n\n1. Identity Providers → Create → GitHub\n2. Paste:\n   - Client ID\n   - Client Secret\n3. Save\n\n## 4. Test\n\n1. Go to http://localhost:3000/api/oauth/login\n2. On Keycloak page, see \"Google\" and \"GitHub\" buttons\n3. Click either button\n4. Login with that provider\n5. Should be redirected back to your app\n\n## Done! ✅\n\nUsers can now login with Google or GitHub.\n\n## Next Steps\n\n- See `KEYCLOAK_SOCIAL_LOGIN.md` for advanced configuration\n- Enable user account linking\n- Customize login page\n- Setup event logging\n
